# 🎓 Student Portal System

A full-stack web application that simulates a college ERP system, allowing students to manage academic and administrative information in one place.

---

## 🚀 Features

* 👤 Student Profile Management
* 📚 Course Registration
* 📊 Marks & GPA Tracking
* 📅 Timetable Management
* 🧾 Fee Status Tracking
* 📝 Exam Schedule
* 🔔 Notifications & Announcements
* 💬 Feedback System

---

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Data Storage: JSON-based local storage

---

## 📂 Project Structure

* index.html — Frontend UI
* style.css — Styling
* script.js — Frontend logic
* server.js — Backend API
* data.json — Data storage

---

## ▶️ How to Run

### 1. Install dependencies

```bash
npm install
```

### 2. Start server

```bash
node server.js
```

### 3. Open in browser

```
http://localhost:3000
```

---

## 📡 API Endpoints

* `/api/student` — Get/update student details
* `/api/subjects` — Get subjects
* `/api/registered` — Manage course registration
* `/api/marks` — View/update marks
* `/api/attendance` — Attendance data
* `/api/fees` — Fee details
* `/api/exams` — Exam schedule
* `/api/notifications` — Notifications
* `/api/feedback` — Submit feedback

---

## 🎯 Purpose

This project demonstrates how a student portal system works by combining frontend UI with backend APIs, helping in understanding full-stack development concepts.
